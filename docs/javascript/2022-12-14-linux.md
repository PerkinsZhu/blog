## 查看内存
- free
- free -m 查看多少MB
- free -g 查看多少GB

## 查看文件（可以打印多个文件）
- cat file
- head file // 开头的行数
- head -n 5 file  // 开头五行
- tail file  // 结尾的行数
- tail -n 5 file // 结尾5行
- wc -l file  // 统计文件行数
