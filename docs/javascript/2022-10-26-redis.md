- 简单安装 启动 连接  配置 使用
  - https://juejin.cn/post/6979019298543140901#heading-5
  
- 查看版本
```
redis-server - v
redis-server --version
```

- 查看redis运行的进程
```
ps -aux | grep redis
ps -ef|grep redis
可以看到redis安装的目录
```

- 启动
```
redis-server redis.conf
```

- 修改redis.conf
```
//修改密码
requirepass  password

//开启守护
daemonize  yes

// 修改后重启
redis-server redis.conf
``` 


- 进入redis
```
redis-cli

auth password
// 成功后进入
```

- 使用命令
```
SET key
GET key
```
