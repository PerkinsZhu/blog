- 子组件中的代码
  immediate 默认值为 false,则第一次不会监听 watch 的执行
  immediate 设置为 true,则第一次也会监听 watch 的执行
  watch 位于子组件中，子组件通过 v-if 和 v-show 控制显示时，

```javascript
watch(
  () => props.changeName,
  (nVal, oVal) => {
    console.log(nVal, oVal, "watch-changeName");
  },
  {
    immediate: true,
    deep: true,
  }
);
```
